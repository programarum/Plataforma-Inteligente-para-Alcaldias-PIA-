# ADR-0001: Modular Monolith

- Status: Accepted
- Date: 2026-07-01

## Context

El proyecto necesita una base sólida, clara y escalable sin introducir complejidad innecesaria en las primeras etapas.

## Decision

Se adoptará una arquitectura de modular monolith como punto de partida para mantener la simplicidad operativa y favorecer la evolución futura.

## Consequences

- Se reduce la complejidad inicial.
- Se facilita el crecimiento incremental del producto.
- La base es adecuada para posteriores separaciones de módulos si el producto lo requiere.

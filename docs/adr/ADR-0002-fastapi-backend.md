# ADR-0002: FastAPI Backend

- Status: Accepted
- Date: 2026-07-01

## Context

Se requiere una base backend moderna, clara y preparada para crecer, manteniendo el enfoque inicial en la estructura del repositorio.

## Decision

Se documenta la intención de usar FastAPI como tecnología principal para el backend, sin implementarlo aún en esta fase.

## Consequences

- Se deja una ruta clara para construir la API de manera consistente.
- Se preserva la posibilidad de evolucionar con documentación y contratos bien definidos.

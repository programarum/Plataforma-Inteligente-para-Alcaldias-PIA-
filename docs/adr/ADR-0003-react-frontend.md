# ADR-0003: React Frontend

- Status: Accepted
- Date: 2026-07-01

## Context

Se necesita una interfaz moderna y modular para la futura experiencia de usuario del producto.

## Decision

Se documenta el uso de React como tecnología base para el frontend, sin implementarlo todavía.

## Consequences

- Se establece una dirección clara para la capa de experiencia de usuario.
- Se mantiene el proyecto alineado con una arquitectura de interfaces moderna y escalable.

# ADR-0005: AI Gateway

- Status: Accepted
- Date: 2026-07-01

## Context

La plataforma futura podrá incorporar capacidades de inteligencia artificial y recuperación de información, por lo que es conveniente preparar una estrategia inicial.

## Decision

Se definirá un gateway de IA como punto de integración para futuras capacidades cognitivas y de búsqueda, sin implementar aún la funcionalidad.

## Consequences

- Se deja una ruta clara para integrar capacidades de IA de forma controlada.
- Se preserva la modularidad del producto para futuras extensiones.

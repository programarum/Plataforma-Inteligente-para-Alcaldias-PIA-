# ADR-0004: PostgreSQL Database

- Status: Accepted
- Date: 2026-07-01

## Context

El producto requerirá almacenamiento estructurado, integridad relacional y compatibilidad con cargas futuras de negocio.

## Decision

Se adoptará PostgreSQL como base de datos principal para el proyecto, sin implementarla aún en esta etapa.

## Consequences

- Se establece una base sólida para datos transaccionales y relacionales.
- Se facilita la integración con futuras capas de negocio y servicios.

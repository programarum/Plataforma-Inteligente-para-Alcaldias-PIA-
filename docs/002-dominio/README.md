# Dominio de PIA

El dominio representa las capacidades institucionales que PIA apoyara. Durante esta fundacion solo se establecen areas de exploracion; no se definen entidades, agregados ni reglas de negocio definitivas.

## Areas iniciales

- Identidad, acceso y seguridad.
- Gestion y recuperacion de conocimiento institucional.
- Busqueda y consulta documental.
- Asistencia mediante inteligencia artificial.
- Auditoria y trazabilidad.
- Integraciones y canales de experiencia.

## Modelado

Cada capacidad debe modelarse con especialistas del dominio. El lenguaje ubicuo, los limites modulares y las invariantes se documentaran antes de codificarse. Las dependencias entre modulos deben ser explicitas y respetar la arquitectura aprobada.

## Estado

Pendiente de talleres de descubrimiento y definicion funcional. Este documento no autoriza implementaciones ni presupone reglas institucionales.

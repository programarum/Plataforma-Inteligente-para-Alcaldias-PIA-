# Código de Conducta

## Nuestro compromiso

Nos comprometemos a hacer de la participación en este proyecto una experiencia libre de acoso para todas las personas, independientemente de edad, identidad, discapacidad, apariencia, origen étnico, experiencia o condición socioeconómica.

## Comportamiento esperado

- Respetar a todas las personas involucradas.
- Mantener un ambiente profesional y colaborativo.
- Aceptar críticas constructivas de manera respetuosa.
- Priorizar el bienestar del proyecto y del equipo.

## Comportamiento inaceptable

- Comentarios ofensivos o discriminatorios.
- Acoso, intimidación o amenazas.
- Publicación de información privada sin permiso.
- Conductas que afecten negativamente el entorno colaborativo.

## Aplicación

Las conductas inapropiadas podrán ser reportadas al equipo del proyecto y serán revisadas de manera oportuna.
